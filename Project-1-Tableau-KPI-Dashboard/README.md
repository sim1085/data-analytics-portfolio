# Project 1: E-commerce Sales KPI Dashboard (Tableau)

## Overview
Built an executive-level Tableau dashboard to analyze sales performance for an e-commerce dataset (99K+ orders).

## KPIs Included
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value (AOV)
- Monthly Revenue Trends
- Top Product Categories
- Revenue by State
