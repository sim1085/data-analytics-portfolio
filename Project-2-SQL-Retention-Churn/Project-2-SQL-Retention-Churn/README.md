# Project 2: Customer Retention & Churn Analysis (SQL)

## Objective
Analyze customer repeat purchase behavior and churn using PostgreSQL, similar to product analytics workflows in Amplitude/Mixpanel.

## Dataset
Olist Brazilian E-commerce Dataset  
- 99K+ orders  
- 96K+ customers  

## Key Metrics
- Repeat customers: 2,997  
- Repeat rate: 3.12%  
- Churn rate: ~96.88%  

## Analysis Performed
- Repeat customer segmentation
- Retention & churn definitions using MIN/MAX
- Advanced SQL: CTEs, window functions, lifecycle metrics

## Business Insight
Customer retention is extremely low, indicating strong opportunity for lifecycle marketing and re-engagement strategies.
